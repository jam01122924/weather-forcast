Introduction to Weather Forcast Demo

Weather Forcast Demo

This project was developed using Angular2 with angular-cli version 1.0.0-beta.17.

The website is designed and developed by James Le at 2016 Oct

Environment setup

Firstly, you will need to install NPM and Git in order to make it work.

Once you have installed NPM, all you will have to do is run the following commands:

npm install

Runing server

Run the following commands for a dev server:

ng serve

Navigate to http://localhost:4200/. The app will automatically reload if you change any of the source files.

Run the following commands to build the project:

ng build

The build artifacts will be stored in the dist/ directory.

For production build, use the -prod flag.

Running unit tests

Run the following commands to execute the unit tests via Karma:

ng test
